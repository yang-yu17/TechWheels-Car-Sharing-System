# TechWheels Car Sharing System

## Overview

TechWheels is a vehicle-sharing management system designed for a university campus environment. The project was developed using MySQL and Python to manage members, vehicles, reservations, trips, maintenance records, and incident reports.

The system provides a centralized database solution that supports efficient vehicle reservation and operational management.

---

## Features

* Member Management
* Vehicle Management
* Reservation Tracking
* Trip Monitoring
* Maintenance Record Management
* Incident Reporting
* Staff and Hub Administration

---

## Database Design

The database was designed using relational modeling principles and normalized to Third Normal Form (3NF).

Main entities include:

* Members
* Membership Tier
* Membership Card
* Vehicle
* Reservations
* Trip
* Maintenance Record
* Incident Report
* Staff
* Hub

The design includes primary keys, foreign keys, referential integrity constraints, and relationship modeling for real-world business operations.
